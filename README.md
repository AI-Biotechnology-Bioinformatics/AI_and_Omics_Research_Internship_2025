## Tasks and Assignments
**Class 3CB:** Differential Gene Expression Analysis   
> + Map probe IDs to gene symbols using AnnotationDbi (Find the appropriate annotation package for your platform and use it. Check how many probes map to the same gene and handle duplicates)
> + Perform differential gene expression analysis using the Limma package. Save DEG results (complete, upregulated, downregulated) as CSV files
> + Create a volcano plot showing upregulated and downregulated genes. Generate a heatmap of the top 25 DEGs. Export both plots as PNG images in the Results folder
> + Write a short summary of your results. Explain: How many multiple probes mapped to the same gene and how you handled duplicate probes.State which contrast or comparison you performed for DEGs analysis. (e.g cancer_vs_normal etc) and summarize how many genes were upregulated and downregulated based on your DEG results.
